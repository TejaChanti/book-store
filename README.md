# BookStore

Users can Create, Read, Update or Delete 'Books' data from the database.

## Tech Stack

- **Server:** Node JS, Express JS
- **Database:** MongoDB


## Run Locally

### Clone the project

```bash
git clone https://github.com/TejaChanti/book-store.git
```

### Install dependencies

```bash
npm install
```

### Start the server
```bash
npm start

```
